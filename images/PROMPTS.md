## 3. Character Design Rules (قواعد تصميم الشخصيات)

### 3.1 General Rules
- All figures wear **full Islamic dress**: thobe (long garment) + ghutra/kufi (head covering)
- No exposed skin except face and hands
- Faces should be **round, friendly, child-appropriate** (big eyes, soft features — anime/kids' book style)
- Children should look **visibly young** (shorter stature, rounder faces, smaller limbs compared to adults)
- Adults appear taller, with dignified bearing

### 3.2 Character Consistency
- Each main child character must have a **consistent appearance** across all their scenes:
  - Same thobe color
  - Same head covering style/color
  - Same approximate height/build
- Background/extra characters can vary freely

## 4. Environment Style (البيئة البصرية)

### 4.1 Desert Landscape (الصحراء)
- Golden sand dunes with soft curves
- Date palms (palm trees) — slender trunks, green fronds
- Rocky outcrops in warm browns
- Horizon with gradient sky (day: gold/blue, sunset: orange/purple, night: deep blue)

### 4.2 Madinah Architecture (المدينة المنورة)
- Mud-brick buildings in warm brown/beige tones
- Flat roofs, small windows, arched doorways
- Minaret silhouettes
- Date palm gardens
- Simple mosque structure with dome

### 4.3 Interiors (الداخلية)
- Tent interiors: woven fabric walls, carpeted floor, oil lamps, cushions
- Study areas: rehal (book stand), parchment, inkwell, lamp
- Simple furniture, no anachronisms (no modern objects)

### 4.4 Night Scenes (المشاهد الليلية)
- Starry sky (variable star positions per scene)
- Crescent moon
- Campfire glow
- Lamp-lit tents

## 6. Diversity & Non-Repeat Rules (قواعد عدم التكرار)

To keep children engaged, **EVERY SCENE MUST BE VISUALLY UNIQUE:**

| Rule | Description |
|------|-------------|
| **No identical backgrounds** | Each scene must have a different background composition, time of day, or angle |
| **Vary lighting** | Alternate between: golden hour, night, sunset, dawn, lamp-lit, shade |
| **Vary camera distance** | Mix: wide shots, medium shots, close-ups, extreme close-ups |
| **Vary character poses** | No two scenes with the same pose for the same character |
| **Unique colour emphasis** | Each scene should have a different dominant colour accent |
| **Seasonal/natural variation** | Desert can look different: dunes, rocky, oasis, dusty, clear sky |
| **Avoid repeated props** | Don't reuse the same tent, palm tree arrangement, or lamp position |

### Scene Diversity Checklist
- [ ] Background type: desert / mosque / tent / city / night sky / battlefield / study
- [ ] Lighting: sunrise / noon / sunset / night / lamp / fire / divine glow
- [ ] Shot type: wide / medium / close-up / over-shoulder / bird's eye
- [ ] Number of figures: solo / duo / small group / crowd
- [ ] Mood: serious / joyful / emotional / contemplative / action
- [ ] Dominant colour: gold / blue / orange / green / brown / white

> ✅ **Every scene in the prompts above already follows these rules.**
> When generating, ensure your selected scene matches the description **exactly**
> and does not duplicate the composition of any other scene.

## 7. Technical Generation Notes

### Recommended Settings (Leonardo.ai)
- **Model:** Leonardo Anime XL or DreamShaper (for anime storybook style)
- **Aspect Ratio:** 16:9 (--ar 16:9)
- **Resolution:** Up to 1024×576 or higher
- **Guidance Scale:** 7-9
- **Steps:** 30-40
- **Style:** Cinematic, Storybook, Warm

### Character Placement
- Avoid putting characters in the exact center of every frame
- Use rule of thirds for more cinematic composition
- Leave some space at top/bottom for text overlays (the app will add text over the images)

### Prohibited Elements (ممنوعات)
- ❌ No modern clothing, objects, or architecture
- ❌ No exposed awrah (intimate body parts) — all figures fully dressed
- ❌ No faces of the Prophet ﷺ (he should be shown from behind, in silhouette, or with his face obscured by light)
- ❌ No depictions of angels
- ❌ No violent/gory details — child-appropriate only
- ❌ No anachronisms (no electricity, cars, etc.)
- ❌ No repeated images across different scenes
